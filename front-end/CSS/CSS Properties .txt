#CLASS SELECTOR
.red-text{                         .class{
    color:red;                            property: value
    }                                    }

<h2 class="red-text">Red</h2>
<h2>Green</h2>
<h2>Blue</h2>


Now in the css code, it selects the only eleemt that has class red text and apply the property to the Element

#ID SELECTOR

It is applicable to only specific element in a html file
Wheras class selector is applicable to multiple elements

HTML FILE
<h2 id="main">Red</h2>
<h2>Green</h2>
<h2>Blue</h2>

CSS FILE
#main{
    color: red;
}
